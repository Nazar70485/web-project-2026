# TechHub - Projekt Aplikacji Webowej

## Cel projektu
Stworzenie w pełni funkcjonalnej aplikacji webowej demonstrującej umiejętności z zakresu nowoczesnych technologii (HTML5, CSS3, JavaScript).

## Funkcjonalności
* **Responsywny Design:** Interfejs dostosowany do urządzeń mobilnych i desktopowych (Flexbox, Grid).
* **3 Podstrony:** Strona Główna, Projekty, Kontakt.
* **Integracja z API:** Dynamiczne pobieranie danych za pomocą Fetch API.
* **Interakcja:** Walidacja formularza kontaktowego i obsługa zdarzeń DOM.

## Technologie
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS).
* **API:** JSONPlaceholder (symulacja danych projektowych).
* **Kontrola wersji:** Git / GitHub.

## Instrukcja uruchomienia
1. Sklonuj repozytorium: `git clone https://github.com/Nazar70485/web-project-2026.git`
2. Otwórz plik `index.html` w dowolnej przeglądarce.
